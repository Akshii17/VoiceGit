
Command: git push origin main
Explanation: Uploads your commits to the remote repository

Command: git add .
Explanation: Stages your changes (prepares files to be committed)

Command: git commit -m 'auto'
Explanation: Saves a snapshot of your project locally

Command: git status
Explanation: Shows working tree status (staged, modified, untracked)

Command: git checkout -b Trisha
Explanation: Creates a new branch and switches to it

Command: git log
Explanation: Shows commit history

Command: git commit -m "added all files"
Explanation: Saves a snapshot of your project locally
